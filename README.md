# Flood-it/Flood Fill
The objective of the game is to fill the board with one color within 25 moves. You select a color by clicking on it, and it will fill from the top left color.

## Compiling

You will need:
* C compiler
* Raylib
* make

To Compile:
```
git clone "https://github.com/acly7946/Flood-It"
cd directory
make
```
